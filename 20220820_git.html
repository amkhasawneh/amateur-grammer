Yesterday, I finished the Git and GitHub course by Colt Steele on Udemy. (I highly recommend this course, by the way.) Would that I'd known earlier what a great tool Git is! Every since I started learning it, I have been using it at work, for downstream single-cell RNA sequence (scRNA-seq) analysis, mainly using the R language. Maybe I should write more about that in detail later.

To me, the most interesting thing in Git, apart from the real work it does, is that it gives me a sense of achievement, and a way to mark progress. I have been also pushing my work on to GitHub (privately) to get a better understanding and experience with remote repositories. I am around 25% into the Pro Git book (free and good), and like it so far. So I guess the important thing is that I am gaining skills, and, importantly, have the opportunity to put them to (good) use.
